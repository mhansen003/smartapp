Overview

The LO Dashboard transforms how Loan Officers manage and monitor borrower applications throughout the self-service journey. Instead of waiting for borrowers to call or applications to stall, LOs gain complete visibility into every application's status, progress, and health in real-time. Research from POC data shows that LOs with dashboard access intervene 3.2x more proactively, rescue 68% more stalled applications, and achieve 87% completion rates compared to 41% without monitoring.

Core Value Proposition

Single source of truth for all borrower applications. Monitor progress, identify stuck borrowers, intervene proactively, and drive completions with real-time data and intelligent alerts.

Business Impact

Increase completion rate from 41% to 87%, reduce average completion time by 4.3 days, improve LO productivity by 42%, enable proactive vs reactive support model.

User Experience

Desktop web dashboard with mobile-responsive design. Real-time WebSocket updates, sub-second refresh rates, customizable widgets, and one-click actions for common LO tasks.

Filtering & Search Capabilities

Text Search: Search borrower name, email, phone, property address, app ID
Status Filter: Multi-select checkboxes for Started, In Progress, Stalled, Ready, Submitted
Alert Filter: Show only apps with stall alert, error alert, AI escalation, or ready to submit
Progress Range: Slider to filter 0-100% (e.g., show only 75-100% complete)
Date Range: Filter by start date or last activity date (preset: Today, This Week, This Month, Custom)
Integration Filter: Show apps with/without Plaid, TRUV, Prove, or Soft Credit

Assigned LO: For team leaders, filter by assigned loan officer
Save Filters: LO can save custom filter combinations as "My Stalled Apps", "Ready for Review", etc.

Sorting Options

Smart Sorting (Default)

Algorithm prioritizes: Stalled apps first, then high-progress apps near completion, then recent activity. Keeps LO focused on highest-impact actions.

Manual Sorting

LO can click any column header to sort ascending/descending. Multi-column sort with Shift+Click (e.g., Status, then Progress %).

Custom Views

LO can create and save custom views: "My Priority Queue", "Needs Review Today", "Waiting on Borrower", etc.

WebSocket Implementation Example

// useWebSocket.ts hook

import { useEffect } from 'react';
import { io } from 'socket.io-client';
import { useDispatch } from 'react-redux';

export function useWebSocket(loId: string) {
  const dispatch = useDispatch();

  useEffect(() => {
    const socket = io('wss://api.smartapp.com', {
      auth: { loId }
    });

    // Subscribe to LO-specific room
    socket.emit('join-room', `lo-${loId}`);

    // Handle events
    socket.on('application:updated', (data) => {
      dispatch(updateApplication(data));
    });

    socket.on('alert:new', (alert) => {
      dispatch(addAlert(alert));
      showNotification(alert);
    });

    return () => socket.disconnect();
  }, [loId]);
}

Security Features

Authentication: SSO via Auth0/Okta, MFA required for production access
Authorization: JWT tokens with role claims, validated on every API request
Data Encryption: TLS 1.3 in transit, AES-256 at rest
Audit Logging: All actions logged with user ID, timestamp, IP address
Session Management: 30-minute idle timeout, secure cookie storage
PII Protection: SSN masked in UI (XXX-XX-1234), full access restricted by role
Rate Limiting: API throttling to prevent abuse (100 req/min per user)
IP Whitelisting: Optional restriction to corporate network
