URLA Section-by-Section Coverage

Borrower Information (Section 1)

FlueID Coverage with Address Only: 0 fields

Borrower personal information (Name, SSN, DOB, Employment, Income) is not available in property records. With SSN (Credit Bureau), we can potentially verify name spelling, DOB, SSN, and see employment history.

Strategic Recommendations

Recommended FlueID Use Cases for Smart App 2.0

1. Refinance Flows: Use title data to auto-identify co-borrowers on title and existing liens
2. REO Section: Validate investment properties against title records for accurate data
3. Data Prefill: Auto-populate property details, mortgage info, and tax/HOA data
4. Risk Flagging: Identify bankruptcy, foreclosure, or judgment indicators for LO review
5. Cost Efficiency: Leverage flat-fee model ($5k/month unlimited runs) for enhanced data enrichment
