Overview

The Uniform Residential Loan Application (URLA) 2020 is the standardized form used by mortgage lenders. This gap analysis identifies which URLA fields are collected in Smart App 2.0, which are gaps that need to be addressed, and how data flows from the borrower experience through to loan origination systems.

Purpose

Ensure complete URLA compliance by identifying all required fields and mapping them to Smart App data collection points.

Scope

Covers all sections of URLA 2020 including borrower information, property details, financial data, and declarations.

Status

Complete field matrix available. Integration mapping in progress with data validation requirements defined.
