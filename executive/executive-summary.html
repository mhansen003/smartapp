Project Vision

Transform the traditional mortgage application from a long, repetitive form into a streamlined intake followed by gamified task completion

Smart App 2.0 reimagines the digital mortgage experience by separating the initial data capture from the completion workflow. Borrowers provide essential information quickly, then complete verification tasks one-at-a-time in a guided, intelligent flow powered by third-party integrations and AI assistance.

Top 5 Strategic Priorities

1. Task Workflow System (P0 - Highest Priority)

Why: Core differentiator from traditional apps. Transforms completion experience.
What: One-at-a-time guided task flow with "5 of 7 complete" gamification
Impact: Reduces borrower overwhelm, increases completion rates

2. Prove Identity Verification (P0)

Why: Instant identity verification reduces fraud, improves UX
What: Phone number verification with instant identity confirmation
Impact: 90% success rate, eliminates manual ID collection

3. FlueID Title Data Integration (P0)

Why: Auto-populate 22-40 URLA fields, massive time savings
What: Property data extraction from title records
Impact: 50% reduction in manual data entry for refinances

4. Mobile AI Chat Experience (P1)

Why: Most borrowers start on mobile, need instant help
What: Context-aware AI assistant throughout journey
Impact: Reduces LO support burden, improves borrower confidence

5. LO Configuration Dashboard (P1)

Why: LOs need control over automation vs manual workflows
What: Toggle switches for each automation feature
Impact: Empowers LOs to choose their preferred balance
